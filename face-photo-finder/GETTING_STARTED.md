# Simplified Face Photo Finder - No Backend

> Find yourself in event photos using AI face recognition - runs entirely in your browser!

## 🚀 Ultra-Fast Setup

```bash
cd face-photo-finder
npm install
npm run dev
```

Visit: http://localhost:3000

## 📦 Deploy in 1 Minute

```bash
# Build
npm run build

# Push to GitHub
git add .
git commit -m "Deploy Face Photo Finder"
git push origin main

# Done! Your app is live on GitHub Pages
```

**URL:** https://yourusername.github.io/portfolio/face-photo-finder/

## ✨ What's Included

- ✅ Upload ZIP of photos

- ✅ Camera capture support
- ✅ Download matched photos
- ✅ 100% runs in browser
- ✅ Zero backend needed

## 📋 Requirements

- Node.js 18+
- Modern browser
- Internet (for loading AI models once)

## 🎯 How to Use

1. **Upload a ZIP** of your event photos
2. **Upload your selfie** or capture with camera
3. **App finds you** in the photos!
4. **Download** matched photos

## 🔐 Privacy

- All processing happens **locally** in your browser
- Photos never uploaded anywhere
- Zero data tracking
- Completely private

## 📊 Performance

- Works with 100-1000 photos (depends on your device)
- Takes 1-2 seconds per photo to process
- Instant search once indexed
